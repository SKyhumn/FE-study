# HTML기본 구성 요소

## HTML의 기본 구조

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
    
    </body>
</html>

<head>는 문서 정보를 정의한다.
<body>는 웹 브라우저에 표시할 내용을 담는다.
