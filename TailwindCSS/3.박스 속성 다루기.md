# 3.박스 속성 다루기

## 너비 및 높이 조정

```html
<div className="w-32 h-32"></div>
```

- `w-*` :너비
- `h-*` :높이
- 너비 및 높이가 1씩 늘어날 때마다 0.25rem씩 너비 높이가 늘어난다.

### 최대 너비 높이 및 최소 너비 높이

```html
<div className="max-w-sm max-h-sm"></div>
<div className="min-w-sm min-h-sm"></div>
```

- `max-w-*` :최대 너비
- `max-h-*` :최대 높이
- `min-w-*` :최소 너비
- `min-h-*` :최소 높이

`max/min-w/h-*` 은 `sm` , `m` 등으로 크기를 조절한다.

**max/min-w/h-값**

| **max/min-w/h-*** | **rem** |
| --- | --- |
| `xs` | 20rem |
| `sm` | 24rem |
| `md` | 28rem |
| `lg` | 32rem |
| `xl` | 36rem |
| `2xl` | 42rem |
| `3xl` | 48rem |
| `4xl` | 56rem |
| `5xl` | 64rem |
| `6xl` | 72rem |
| `7xl` | 80rem |

## Padding

```html
<div className="p-1"></div>
```

- `p-*` :전체 패딩 값 지정

### 방향별로 Padding 지정

```html
<div className="px-1 py-1"></div>
```

- `px-*` :좌우 패딩
- `py-*` :상하 패딩

```html
<div className="pt-1 pb-1 pl-1 pr-1"></div>
```

- `pt-*` :상 패딩
- `pb-*` :하 패딩
- `pl-*` :좌 패딩
- `pr-*` :우 패딩

## Margin

```html
<div className="m-1"></div>
```

- `m-*` :마진 값 지정

### 음수 마진도 가능

```html
<div className="-m-1"></div>
```

실무에서 카드 겹칠 때 사용

### 방향별로 마진 지정

```html
<div className="mx-1 my-1"></div>
```

- `mx-*` :좌우 마진
- `my-*` :상하 마진

```html
<div className="mt-1 mb-1 ml-1 mr-1"></div>
```

- `mt-*` :상 마진
- `mb-*` :하 마진
- `ml-*` :좌 마진
- `mr-*` :우 마진

### `auto` 사용하기

- `auto`는 `flex` 인 부모에 남는 공간이 있을 때, 그 공간을 가져가서 요소를 밀어주는 값이다.
- 마진 속성에만 사용할 수 있다.

```html
mx-auto
ml-auto
mr-auto
mt-auto
mb-auto
```

## Border

### Border를 키는 기본 값

```html
<div className="border"></div>
```

### Border 두께 정하기

```html
<div className="border-1"></div>
```

- `border-*` :보더 두께 지정
- 숫자는 px 단위다. (4=4px)

### 방향별 Border 지정해주기

```html
<div className="border-x-1 border-y-1"></div>
```

- `border-x-*` :좌우 Border 지정
- `border-y-*` :상하 Border 지정

```html
<div className="border-t-1 border-b-1 border-l-1 border-r-1"></div>
```

- `border-t-*` :상 Border
- `border-b-*` :하 Border
- `border-l-*` :좌 Border
- `border-r-*` :우 Border

### 색상 설정

```html
<div className="border border-red-500"></div>
```

- `border-색-진하기` 로 지정
- 투명도도 지정 가능

**투명도**

```html
<div className="border border-black/20"></div>
```

- Border의 색이 검은 색일 때만 가능하다.

### 모서리 둥글기

```html
<div className="border rounded"></div>
```

- 모서리를 둥글게 해준다.
- 모서리 둥글기도 `xs` ~ `full` 까지 둥글기 정도 지정이 가능하다. (예: `rounded-xl`)
- 모서리 둥글기도 방향 지정이 가능하다. (예: `rounded-t-xl` , `rounded-r-l`)

### Border 제거/리셋

```html
border-0
border-none
```

### 쉐도우 지정해주기

```jsx
<div className="shadow-md"></div>
```

- `xs` , `md` , `xl` 등으로 그림자 크기를 지정해줄 수 있다.
- 속성 값을 `inner` 로 하면 눌린 느낌이 난다.

```jsx
<div className="shadow-md shadow-red-500/30"></div>
```

- 쉐도우의 색도 지정할 수 있다.
- `shadow-색-진하기` 를 사용하며 `/불투명도` 로 불투명도를 지정할 수 있다.

| **카테고리** | **연결되는 유틸리티** |
| --- | --- |

| `--spacing` | 패딩/마진 유틸 |
| `--border` | 보더 유틸 |
| `--color-border` | 보더 컬러 |
