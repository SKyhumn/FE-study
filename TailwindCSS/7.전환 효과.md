# 7.전환 효과

## transition

```html
<div className="transition"></div>
```

- Tailwind는 이미 **transition 유틸리티**를 다 준비해놨다.

## transition-*

```html
<div className="transition-all"></div>
<div className="transition-colors"></div>
<div className="transition-transform"></div>
```

- `transition-*` : 무엇을 전환 시킬건지 지정해줄 수 있다.

## transition duration-*

```html
<div className="transition duration-500"></div>
```

- `transition duration-*` :전환 지속 시간을 얼마나 할 것인지 정할 수 있다.

## transition delay-*

```html
<div className="transition delay-500"></div>
```

- `transition delay-*` :몇 ms 후 전환을 진행할 것인지 정해준다.

## 가속도 (easing)

```html
<div className="transition ease-in-out"></div>
```

```html
<div className="transition ease-[cubic-bezier(0, 0, 0, 1.03)]"></div>
```

- 어떤 방식으로 전환을 진행할 것인지 정할 수 있다.

## transition 작동 시키는 법

```html
<div className="
	w-25 
	h-25 
	bg-red-500 
	transition-colors 
	transition duration-500 
	transition ease-[cubic-bezier(0.25, 1, 0.5, 1)]
	 
	hover:bg-blue-500" 
>
	마우스 커서가 올라왔을 때 색 변경
</div>
```

- transition 속성 값들을 지정해준 뒤 가상 클래스를 만족 했을 때 바뀔 스타일을 지정한다.
- 위 코드는 `bg-red-500` 였다가 마우스 커서가 올라왔을 때 `bg-blue-500` 으로 바꿔준다.

## @theme에서 쓸 수 있는 카테고리

| **카테고리명** | **연결되는 유틸리티** |
| --- | --- |
| `--transition-property` | `transition-*` |
| `--transition-duration` | `transition duration-*` |
| `transition-delay` | `transition delay-*` |
| `transition-timing-function` | `transition ease-*` |
