# 6.가상 요소, 가상 클래스 사용

## 가상 요소

`::before` , `::after` 지원

```html
<div className="before:content-['']"></div>
```

```html
<div className="after:content-['']"></div>
```

- 요소의 앞쪽이나 뒤쪽에 콘텐츠를 놓을 때 사용한다.
- `before` :요소 앞 쪽에 콘텐츠
- `after` :요소 뒤 쪽에 콘텐츠
- `content-*` 가 반드시 붙어야 한다.

## 가상 클래스

```html
<div className="hover:bg-red-500"></div>
```

- 요소에 상태에 따라 스타일을 적용시킨다.
- `hover` :마우스를 올렸을 때
- `active` :클릭 중
- `focus` :포커스 되었을 때
- `disabled` :비활성화 되었을 때
