# 8.애니메이션

keyframe을 쓰기 위해선 CSS파일 내부에서 `@keyframe` 을 만들어 줘야 한다.

```css
@keyframe changing-color{
	0%{/*코드*/}
	n%{/*코드*/}
	100%{/*코드*/}
}
```

이후 `@theme` 에서 `--animate` 카테고리를 이용해 디자인 토큰을 정의한다.

```css
@theme{
	--animate-changing-color:<name> <duration> <timing-function> <delay> <iteration-count> <direction> <fill-mode> <play-state>
}
```

`animate-*` 유틸을 이용하여 애니메이션을 적용시킨다.

```html
<div className="text-red-500 animate-changing-color"></div>
```
