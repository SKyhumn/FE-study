# 5.위치 속성

## position 속성

```html
<div className="static">static</div>
```

- tailwind css는 CSS의 position 속성을 그대로 클래스화 했다.
- `static` :요소를 기본 흐름에 따라 배치한다.
- `relative` :요소의 **원래 위치를 기준**으로 **상대적으로(relative)** 이동시킨다.
- `absolute` :가장 가까운 **포지션이 지정된 부모(부모가 relative일 때) 요소 또는 브라우저**를 기준으로 **절대적인(absolute)** 위치에 배치한다.
- `fixed` :절대적인 좌표 위치에 따라 배치하며 스크롤을 해도 위치가 고정되어 있다.
- `sticky` :기본 흐름에 따라 배치하지만, 지정한 좌표의 임계점에 이르면 fixed값처럼 화면에 고정된다.

## 위치 조정

```html
<div className="relative top-20">static</div>
```

- `relative`  혹은 `absolute` 일 때 위치 지정을 할 수 있다.
- `top` :위에서부터 얼마만큼 이동할 건지
- `bottom` :아래에서부터 얼마만큼 이동할 건지
- `left` :왼쪽에서부터 얼마만큼 이동할 건지
- `right` :오른쪽에서부터 얼마만큼 이동할 건지

```html
<div className="relative -top-20 -left-20">static</div>
```

- 음수로도 이동할 수 있다.

## float

```html
<div className="float-left">static</div>
```

- `float` 이란 요소를 왼쪽 또는 오른쪽으로 **“띄워서”** 위치를 조정 및 주변 텍스트/요소가 감싸게 만드는 속성이다.

**float-* 값**

| **속성 값** | **설명** |
| --- | --- |
| `left` | 왼쪽으로 띄운다. |
| `right` | 오른쪽으로 띄운다. |
| `none` | `float` 을 지정해주지 않는다. |

## clear

```html
<div className="clear-left"></div>
<div className="clear-right"></div>
<div className="clear-both"></div>
```

- `float` 을 해제 해준다.

**clear-* 값**

| **속성 값** | **설명** |
| --- | --- |
| `left` | left float 해제 |
| `right` | right float 해제 |
| `both` | 둘 다 해제 |

## z-index

```html
<div className="z-10"></div>
```

- `z-index` 란 요소가 여러개 있을 때 누가 위에 있느냐를 정해준다.
- `z-*` : `z-index` 를 정해줄 수 있다.
- `z-index` 가 클수록 더 위에 있다.

## inset

```html
<div className="inset-4"></div>
```

- `inset` 은 **top / right / bottom / left 를 한 번에 지정하는 축약 속성**이다.
- `inset-*` : `inset` 값을 지정해줄 수 있다.

## @theme에서 쓸 수 있는 카테고리

| **카테고리명** | **연결되는 유틸리티** |
| --- | --- |
| `--spacing` | `top/left/right/bottom/inset-*` |
| `--z` | `z-*`  |
