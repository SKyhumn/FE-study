# 5.위치 속성

## position 속성

```html
<div class="static">static</div>
```

- tailwind css는 CSS의 position 속성을 그대로 클래스화 했다.
- `static` :요소를 기본 흐름에 따라 배치한다.
- `relative` :요소의 **원래 위치를 기준**으로 **상대적으로(relative)** 이동시킨다.
- `absolute` :가장 가까운 **포지션이 지정된 부모(부모가 relative일 때) 요소 또는 브라우저**를 기준으로 **절대적인(absolute)** 위치에 배치한다.
- `fixed` :절대적인 좌표 위치에 따라 배치하며 스크롤을 해도 위치가 고정되어 있다.
- `sticky` :기본 흐름에 따라 배치하지만, 지정한 좌표의 임계점에 이르면 fixed값처럼 화면에 고정된다.

## 위치 조정

```html
<div class="relative top-20">static</div>
```

- `relative`  혹은 `absolute` 일 때 위치 지정을 할 수 있다.
- `top` :위에서부터 얼마만큼 이동할 건지
- `bottom` :아래에서부터 얼마만큼 이동할 건지
- `left` :왼쪽에서부터 얼마만큼 이동할 건지
- `right` :오른쪽에서부터 얼마만큼 이동할 건지

```html
<div class="relative -top-20 -left-20">static</div>
```

- 음수로도 이동할 수 있다.

## float

```html
<div class="float-left">static</div>
```

- `float` 이란 요소를 왼쪽 또는 오른쪽으로 **“띄워서”** 위치를 조정 및 주변 텍스트/요소가 감싸게 만드는 속성이다.

**float-* 값**

| **속성 값** | **설명** |
| --- | --- |
| `left` | 왼쪽으로 띄운다. |
| `right` | 오른쪽으로 띄운다. |
| `none` | `float` 을 지정해주지 않는다. |

## clear

```html
<div class="clear-left"></div>
<div class="clear-right"></div>
<div class="clear-both"></div>
```

- `float` 을 해제 해준다.

**clear-* 값**

| **속성 값** | **설명** |
| --- | --- |
| `left` | left float 해제 |
| `right` | right float 해제 |
| `both` | 둘 다 해제 |

## translate

```html
<div class="translate-x-4"></div>
<div class="translate-y-1/2"></div>
```

- `translate` 는 자기 요소 크기를 기준으로 위치를 지정해준다.
- `translate-x` :자기 너비를 기준으로 좌우 위치 조정
- `translate-y` :자기 높이를 기준으로 상하 위치 조정

```html
<div class="translate-x-4"></div>
<div class="-translate-y-1/2"></div>
```

- 이렇게 음수로도 지정이 가능하다.
