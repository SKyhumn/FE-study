# 4.배경 설정하기

## 배경 색 지정해주기

```html
<div classNeme="bg-red-500"></div>
```

- `bg-색-진하기` :배경 색 정해주기

## 배경 이미지로 설정해주기

```html
<div className="w-screen h-screen bg-[url('/img/bg.jpg')]"></div>
```

- `bg-[url(이미지 주소)]` 로 배경을 이미지로 설정해 줄 수 있다.

## 이미지 반복 설정

```html
<div className="w-screen h-screen bg-[url('/img/bg.jpg')] bg-repeat"></div>
```

- `bg-repeat` :이미지 반복
- `bg-repeat-x` :가로로 이미지 반복
- `bg-repeat-y` :세로로 이미지 반복
- `bg-repeat-round` :이미지를 반복하되 이미지가 요소 크기에 딱 맞도록 크기 조절
- `bg-repeat-space`:이미지를 공간에 맞게 반복
- `bg-no-repeat` :이미지를 반복하지 않음

## 이미지가 요소보다 클 때

```html
<div className="w-screen h-screen bg-[url('/img/bg.jpg')] bg-auto"></div>
```

- `bg-auto` :이미지 크기를 유지한다.
- `bg-cover` :배경 요소에 이미지를 최대한 크게 보여준다.
- `contain` :요소에 이미지를 최대한 크게 보여주되 잘리는 부분이 없게 보여준다.

## 요소 이미지 위치 조정하는 법

```html
<div className="w-screen h-screen bg-[url('/img/bg.jpg')] bg-center"></div>
```

- `bg-center` :사진을 가운데에 둠.
- `bg-right` :사진을 오른쪽에 둠.
- `bg-left` :사진을 왼쪽에 둠.
- `bg-top` :사진을 위에 둠.
- `bg-bottom` :사진을 아래에 둠.

## 삽입된 이미지를 스크롤할 때, 이미지의 작동 방식을 결정

```html
<div className="w-screen h-screen bg-[url('/img/bg.jpg')] bg-scroll"></div>
```

- `bg-local` :삽입된 요소가 모두 스크롤된다.
- `bg-scroll` :삽입된 이미지가 요소에서는 고정되지만, 웹 브라우저에서는 스크롤된다.
- `bg-fixed` :스크롤 해도 이미지가 고정된다.

## @theme에서 쓸 수 있는 카테고리

| **카테고리명** | **연결되는 유틸리티** |
| --- | --- |
| `--color` | `bg-색-진하기`  |
| `--background-image` | `bg-[url(경로)]`  |
| `--background-repeat` | `bg-repeat` |
| `--background-size`  | `bg-*` (이미지 크기) |
| `--background-position`  | `bg-*` (이미지 위치) |
| `--background-attachment`  | `bg-*` (스크롤 처리) |
